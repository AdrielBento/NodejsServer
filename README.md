# 🧛‍♂️ Nodejs API


### Pré requisitos
- Nodejs 10+
- Mysql
### QUICK START
**⚙ Download and install**
```shell
git clone https://github.com/AdrielBento/NodejsServer
cd NodejsServer
npm i
```
**🎲 Create Database**  
Crie um database chamado `mutantes_project` no seu banco de dados Mysql
**Config:**
 - usuario: root
 - password: sem senha
 - server: localhost

**Execute Migrations**
